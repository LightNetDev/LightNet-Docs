This section is for developers who want to understand the internals of the LightNet.


## LightNet SDK

The LightNet SDK is a monorepo that contains the core functionality of the LightNet.
It is written in TypeScript and uses the Astro framework.

Here is a link to the [LightNet SDK repository](https://github.com/lightnetdev/lightnet).
